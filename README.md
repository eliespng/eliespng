# elies.png

Portfolio photographique minimaliste.

## Développement
```bash
npm install
npm run dev
```

## Build
```bash
npm run build
```
Le dossier `dist/` peut être glissé-déposé sur Netlify, ou poussez le dépôt et laissez Netlify exécuter `npm run build` (configuré via `netlify.toml`).

## Photos
Modifiez `src/data/photos.js` pour ajouter/retirer vos photos.
